# Return & Refund Assistant

## Project Description
This project is a simple AI-based chatbot developed for handling return and refund-related customer queries in an e-commerce system.

The chatbot reads information from a policy document and provides relevant responses based on user questions.

---

## Features
- Return policy support
- Refund policy support
- Damaged product support
- Interactive chatbot interface
- Dynamic response from external file

---

## Technologies Used
- Python
- Streamlit

---

## Files in Project
- `app.py` → Main chatbot application
- `policy.txt` → Stores return and refund policies
- `requirements.txt` → Required libraries
- `screenshots/` → Output screenshots

---

### Install Requirements
